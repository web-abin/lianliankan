# 设计图 Prompt 集合 · 低多边形3D治愈风

## 全局风格说明

**核心视觉语言**：本风格定位为 **"低多边形治愈系 3D 卡通风（Cozypunk / Whimsical 3D Cartoon Style）"**，综合以下四大特征：

| 特征 | 描述 | 关键词 |
|------|------|--------|
| **粘土质感** | 所有模型和UI元素看起来像手工捏制的粘土或高质量树脂玩具，扎实圆润，有明显的材质厚度感 | claymation, clay render, plasticine, soft 3D |
| **微缩景观感** | 画面构图类似精美微缩模型盒子（diorama），轻微倾斜俯瞰视角，让玩家有"掌控感"和"安全感" | tilt-shift, diorama, miniature world, top-down isometric |
| **低饱和暖色调** | 色彩饱和度有意压低，以奶油白、燕麦棕、暖橄榄绿、砖红、藕粉为主，营造温暖慵懒的午后氛围 | muted tones, warm desaturated, cozy palette |
| **简化几何自然元素** | 树木简化为团簇状球形几何体，山为圆润多边形，整体降低视觉负担，UI与场景边缘全部做柔化圆润处理 | low-poly, rounded geometry, simplified shapes |

**参考游戏风格**：Monument Valley、Townscapes、Alba: A Wildlife Adventure、Dorfromantik、Unpacking、Kirby and the Forgotten Land（3D段）。

**统一技术规格**：
- 手机竖屏 9:19.5（iPhone 尺寸）
- 渲染感：软3D渲染（soft ambient occlusion），非真实感照明（NPR），顶部柔和漫射光
- 阴影：柔和短圆弧阴影（soft blob shadow），无硬边投影
- 描边：可选极细的轮廓勾线（0~1px，颜色与物体同色系深一档），不强调黑色轮廓
- 色板：奶油白 `#F5EDDC`、燕麦棕 `#C8A882`、暖橄榄 `#8FA874`、砖红 `#C9694A`、藕粉 `#DDB5A8`、天灰蓝 `#A8C4D4`
- UI字体：圆体，厚实，带轻微凸起/浮雕感（emboss），无强烈描边
