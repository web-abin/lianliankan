# 微信小游戏「开心点连连看」界面生图提示词

## 风格说明

本游戏整体视觉风格参考：**清新治愈手绘风**。

核心特征：
- 儿童绘本质感的手绘插画，笔触柔和，线条有温度
- 统一黑色描边，所有角色、UI 元素、按钮均带明显黑色轮廓线
- 配色以薄荷绿、天空蓝、蜜桃橙、奶油白、草地绿为主，整体明快不刺眼
- 卡皮巴拉为主角，圆润胖萌，表情温柔，有治愈感
- 按钮圆润厚实，有轻微立体浮雕感，带黑色描边
- 所有卡片、状态栏采用半透明浅色底板或圆角胶囊形式
- 氛围：治愈、松弛、软糯、儿童绘本、轻喜剧

---

## 通用模版

> 将下方模版与各界面的「界面设计」描述合并，把 `[界面设计]` 替换为对应内容，即可组成完整生图提示词。

```
A mobile casual game UI design illustration in kawaii healing hand-drawn picture book style. Vertical smartphone screen layout, 9:16 aspect ratio. Art style: soft watercolor-like rendering with clean bold black outlines on all elements, warm cheerful palette of mint green, sky blue, peach orange, cream white, and grass green. The mascot is a chubby round capybara with gentle rosy cheeks and soft expressions. UI components feature thick rounded pill-shaped buttons with black outlines, semi-transparent floating card panels, cute coin bag icon and heart icon for resource display. All buttons and icon labels have clear black outline strokes for readability on light backgrounds. No text, no letters, no words anywhere in the image. Illustration only. High quality, clean linework, cozy atmosphere, no clutter.

[界面设计]
```

## 使用说明

1. 复制**通用模版**全文
2. 将 `[界面设计]` 替换为对应界面的描述段落
3. 将组合后的完整文本喂给生图 AI（如 Nanobana）
4. 建议出图尺寸：宽 750px，高 1600px 或以上（全屏页），或 750×900（弹窗类）
5. 弹窗类界面可在提示词末尾补充：`This is a popup dialog floating over a dimmed game background, not a full screen.`
