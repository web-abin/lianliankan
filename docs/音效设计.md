# 皮卡巴拉（治愈系）小游戏音效设计整理

整体风格要求：有机（Organic）、清脆（Crispy）、解压（Satisfying），避免刺耳或机械感，契合水豚泡澡、软萌佛系的氛围。

---

## 1. 核心点击/选中（Selection）
- **风格**：短促、轻盈，不烦躁
- **音效类型**：
  - 泡泡破裂（Bubble Pop / Soap Bubble）：贴合水豚泡澡
  - 木质敲击/块物碰撞（Wooden Click / Block Tap）：呼应木桶、木纹背景
  - 极简系统反馈（Soft UI Tick）

## 2. 成功消除（Match / Eliminate）
- **需求**：突出“任务完成”的满足感
- **音效类型**：
  - 清脆的铃声/风铃（Chime / Bell Gleam）：适配发光特效
  - 闪烁魔法音效（Sparkle / Magic Poof）：与“星星飞向进度条”联动
  - 水滴/水泡声（Plop / Water Drop）：灵动、质感高级

## 3. 连击/成就（Combo & Level Up）
- **建议**：连击音调可逐步升高，加强节奏与爽感
- **音效类型**：
  - 升阶音阶/琶音（Ascending Scale / Arpeggio）
  - 高频叮当（Ting / High-pitched Ding）：标志连击
  - 简短奖励乐句（Successful Reward / Jingle）

## 4. 星星飞行与收集（Travel & Collection）
- **配合**：星星动画、吸入进度条
- **音效类型**：
  - 柔和滑行/飞行音（Whiz / Whoosh Soft）
  - 闪烁拖尾（Zing / Fairy Dust）
  - 星星撞击/收集反馈（Ding Absorption / Item Collect）
