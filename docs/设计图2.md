这是一款治愈系手绘风的游戏首页，采用竖屏中心对称构图。画面核心是一只闭眼泡在木桶里的卡皮巴拉（水豚），头顶橘子，形象憨态可掬。画面风格为干净的 Q 版二次元线条漫，辅以粗厚的深棕色圆润描边。场景布局分为上下两层，由一条深棕色地平线切割；色彩色调以温暖的浅米色、奶油黄和深木色为主，局部点缀橙色与红色。关键元素包括顶部圆角的资源状态栏、两侧垂直对称排列的简笔画风格图标（带文字标签），以及下方一个具有果冻质感的亮橙色大圆角矩形按钮。背景要素是带有微弱纸质纹理的纯色平面，整体观感极简且充满童趣，呈现出一种温馨、休闲的氛围。

# 首页
该界面采用竖屏中心对称布局，视觉锚点聚焦于中部的卡皮巴拉角色插画。1. 空间构图：整体呈“沙漏型”分布，顶部与底部元素密集，中部留白较多以突出品牌。采用垂直分层堆叠逻辑，插画半跨越地平线分割线，增强了空间纵深感。2. 功能划分：顶层状态栏水平排列设置、金币、体力资源槽；中部核心区由粗体描边的主标题和手绘风插画构成，两侧对称分布功能图标（如商店、排行榜等）；中下部交互区为一颗巨大的圆角橙色渐变“开始游戏”按钮，辅以关卡进度说明。3. 关键 UI 形态：所有元素统一为粗厚棕色描边的手绘 Q 版风格。按钮具备高光立体感，侧边栏图标垂直排布并附带文字标签。4. 背景要素：背景为浅米色羊皮纸质感，通过深棕色地平线将画面分为上下两层，确保了前景按钮与插画的视觉稳固性。
