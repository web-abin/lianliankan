这是一款治愈系手绘风的游戏首页，采用竖屏中心对称构图。画面核心是一只闭眼泡在木桶里的卡皮巴拉（水豚），头顶橘子，形象憨态可掬。画面风格为干净的 Q 版二次元线条漫，辅以粗厚的深棕色圆润描边。场景布局分为上下两层，由一条深棕色地平线切割；色彩色调以温暖的浅米色、奶油黄和深木色为主，局部点缀橙色与红色。关键元素包括顶部圆角的资源状态栏、两侧垂直对称排列的简笔画风格图标（带文字标签），以及下方一个具有果冻质感的亮橙色大圆角矩形按钮。背景要素是带有微弱纸质纹理的纯色平面，整体观感极简且充满童趣，呈现出一种温馨、休闲的氛围。

## 全局设计语言

**风格定义**：萌系治愈 Q 版，类 LINE Friends / Kakao Friends 商业插画风格。极简几何形体、强调圆润与笨拙感，传递松弛慢生活氛围。

**色彩体系**（低饱和柔化处理，所有色值均加白/淡化）：
- 主背景色：#FFF8EC（浅米奶油黄，如羊皮纸）
- 地面/分割线：#5C3A1E（深木棕）
- 主按钮色：#FF8C42（温暖橙，果冻高光版）
- 次按钮/强调：#FFB347（浅橙黄）
- 描边色：#3E2010（深棕圆润描边，3-4px，末端圆帽）
- 辅助绿：#9DC08B（嫩草绿，自然元素）
- 辅助蓝：#89C4E1（天空蓝，水/清新）
- 危险/血量红：#E05C5C（柔化红）
- 金币色：#F5C842（暖金黄）
- 文字色：#3E2010（深棕，正文）/ #FFFFFF（白，反底文字）

**描边规范**：所有 UI 元素统一使用 3-4px 深棕色（#3E2010）圆润描边，末端圆帽，转角圆弧过渡，无锐角。

**按钮规范**：所有主要按钮采用"果冻感 3D"效果——顶部弧形高光（白色半透明渐变条）、底部投影（深棕色偏移 3px）、整体圆角矩形（radius 24-32px）；点击时有轻微下压 scale(0.95) + 投影收缩反馈。

**弹窗规范**：所有弹窗采用"IP 异形底板 + bounce 入场"风格。弹窗主体为奶油白圆角大矩形（radius 32px，深棕描边 4px），顶部必须有卡皮巴拉 IP 形象（头部/半身探出弹窗边缘或坐在顶部）。入场动画：scale 从 0.6 弹性放大至 1.05 再回落 1.0，耗时约 300ms。背景：半透明深棕遮罩（#3E2010 @ 50% 透明度）。

**角色设计原则**：卡皮巴拉采用极简几何圆润形体，大圆头、小豆眼（闭眼 = 月牙形/享受状；开眼 = 圆点），棕灰色身体（#8B6C4A），无复杂纹理。配件有橘子（头顶/身边）、小黄鸭（肩膀/旁侧）、白色羊驼（远景/背景）。情绪表达通过极简线条眉毛 + 嘴形变化体现（微笑/撅嘴/星星眼/泪眼）。
