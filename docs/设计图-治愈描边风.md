# 设计图 Prompt 集合 · 治愈描边风

## 全局风格说明

**核心视觉语言**：
- 所有元素必须有清晰的**黑色描边/轮廓线**（2~4px 粗细，手绘质感）
- 画风简洁治愈，圆润可爱，类似治愈系插画游戏（如 Kirby、Animal Crossing）
- 色调温暖柔和：奶油白底色、马卡龙绿、粉橙、淡黄为主
- 卡皮巴拉（水豚）为核心 IP，造型圆滚、呆萌
- 手机竖屏 UI，比例 9:19.5（iPhone 尺寸）
- 字体圆体，按钮圆角 pill 形，UI 元素整体圆润无锐角
