# 设计图 Prompt 集合 · 萌系治愈风

## 全局风格说明

**核心视觉语言**：本风格定位为 **"萌系治愈风格（Cute & Healing Style / Q版清新快消风）"**，以《保卫萝卜》《Cookie Run》《LINE POP》等为代表，综合以下四大特征：

| 特征 | 描述 | 关键词 |
|------|------|--------|
| **"食物化"糖果色** | 饱和度高但极度干净，像马卡龙或糖果表面——每个色值单独提取都像甜点，色与色之间对比明快但不刺激 | candy palette, macaron colors, pastel vivid, clean saturation |
| **极致圆润肉感** | 所有元素胖乎乎、圆滚滚，圆角半径极大，角色像充气气球或软糯团子，视觉上预示"一捏就弹"的Q弹感 | chubby, puffy, inflated balloon feel, round soft body |
| **2D白模光影（Studio AO）** | 虽为2D扁平插画，但在物体边缘和底部用柔和浅色阴影模拟工作室白底打光，让元素具有强烈"实物感"和"可触摸感" | soft studio lighting, 2D ambient occlusion, subtle edge shadow, tangible feel |
| **果冻按钮/极简UI** | UI界面整洁克制，按钮像发光的透明果冻或糖果，信息密度低，焦点集中在角色和交互主体上 | jelly button, clean UI, minimal layout, glow effect |

**参考游戏风格**：保卫萝卜系列、Cookie Run、Candy Crush、LINE POP、切水果、Kirby（2D版）、Pokémon（2D宣传插画）。

**统一技术规格**：
- 手机竖屏 9:19.5（iPhone 尺寸）
- 色板：草莓粉 `#FF8FA3`、马卡龙绿 `#A8E6CF`、柠檬黄 `#FFE66D`、奶油白 `#FFFBF0`、天空蓝 `#A0D8F1`、薰衣草紫 `#C9B8E8`、橘子橙 `#FFB347`
- 描边：细柔描边（1~2px），颜色为物体主色的深一档，不使用纯黑——草莓粉的描边用深玫红，绿色的描边用深绿
- 阴影：2D环境光遮蔽——物体底部和边缘内侧叠加0.5~1px的轻柔白色/亮色内发光（studio highlight），底部对地面落一圈椭圆柔和浅灰软阴影（无硬边）
- 字体：超圆体，字重粗，可爱且高辨识，不使用衬线字体
