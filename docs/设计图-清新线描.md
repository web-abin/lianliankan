这是一款**清新线描极简风**的游戏界面设计方案，在第一版彩铅水彩基础上进行了全面的"减法设计"——去掉繁杂纹理、增加呼吸留白、强化轮廓线条。整体感受从"儿童绘本热闹集市"转向"日系简约插画绘本"：线条更干净、构图更舒展、视线更自然流动。角色依然是春日草地上的卡皮巴拉，但场景更安静、空气感更强。

## 与第一版的核心差异

| 维度 | 第一版（彩铅水彩） | 本版（清新线描） |
|------|------------------|----------------|
| 线条风格 | 3-5px 粗细变化、彩铅抖动 | **2-3px 均匀清晰描边**，端头圆帽，无毛边 |
| 背景纹理 | 斜向彩铅笔触、明显肌理 | **纯色 + 极淡点网格**，干净通透 |
| 装饰密度 | 蒲公英/蝴蝶/小鸭/小花同时出现 | 每个区域**最多 1-2 个装饰元素**，其余留白 |
| 色彩饱和度 | 中高饱和度浓郁配色 | **降低 15-20% 饱和度**，偏柔和粉嫩色调 |
| 元素间距 | 紧凑排布，元素丰富 | 所有行间距 ≥ 24px，卡片 padding ≥ 20px |
| 按钮质感 | 3D 彩铅厚块效果 | **扁平线描按钮**，仅保留 2px 底部偏移阴影 |
| 弹窗装饰 | 头顶卡皮巴拉 + 多种配件 | **卡皮巴拉线描轮廓**，无额外道具，背景大留白 |

## 全局设计语言

**风格定义**：清新线描极简日系插画风。以干净均匀的轮廓线为骨架，低饱和度柔和配色填充，留白作为主要设计语言。画面空旷、通透、如清晨草地上的一本安静绘本。

**色彩体系**（低饱和度柔和春日配色，无纹理叠加）：
- 主背景色：#C8E87A（浅嫩草绿，比 v1 降低约 20% 饱和度）
- 背景次色：#E0F590（超浅草黄绿，光照区）
- 天空/水体色：#7ED4EE（柔和天空蓝）
- 主按钮/强调：#FF9068（浅珊瑚橙）
- 次按钮/辅助：#F7E460（浅蒲公英黄）
- **主描边色（UI/角色）：#1A1A1A（柔和黑，所有卡片、按钮、弹窗、角色轮廓默认描边）**
- **次描边色（草地/自然元素）：#2A5A14（深森林绿，仅用于草地背景笔触、叶片轮廓等自然装饰）**
- 危险/血量：#F07060（柔和暖红）
- UI 底板色：#FFFEF5（云朵白）
- 辅助棕：#A0784A（卡皮巴拉暖棕，比 v1 略淡）
- 文字色：#2A5A14（深森绿）/ #FFFFFF（白色反底文字）

**描边规范**：
- **UI 元素（卡片、按钮、弹窗、图标、进度条）**：统一使用 **2px 柔和黑（#1A1A1A）均匀描边**，干净清晰，线条有重量感但不生硬。
- **角色与动物（卡皮巴拉、小鸭子）**：使用 **2px 柔和黑（#1A1A1A）轮廓**，日系线描标准做法，角色轮廓清晰突出。
- **自然/草地装饰（草地分隔线、叶片、树轮廓）**：使用深森林绿（#2A5A14）描边，与草地背景色协调融合。
- 所有描边末端圆帽，转角平滑圆弧，线条粗细一致，无毛边无抖动。

**按钮规范**：扁平线描按钮——圆角矩形（radius 18-24px）+ 2px 柔和黑（#1A1A1A）均匀描边 + 底部 2px 深色偏移阴影（仅底部，非全3D）。无彩铅纹理，无高光渐变，填色为纯色或极淡双色渐变。点击时有轻微 scale(0.96) + 阴影消失效果（模拟轻压感）。

**点击区域最小尺寸（HitArea）**：与 v1 保持一致，所有可交互元素实际触摸 HitArea 不得小于 **88px 逻辑像素（≈ 176px 设计坐标）**。

**弹窗规范**：云朵白圆角卡片（radius 24px，2px 均匀描边），内部 padding ≥ 28px，弹窗高度的 **30-40% 为空白区域**（不填充内容）。**不再有卡皮巴拉探出边缘的设计**——改为弹窗顶部内侧一个小线描卡皮巴拉图案（约 60×60px，仅线条轮廓 + 简单平涂）。入场动画保留：scale 从 0.7 弹性至 1.0，约 280ms。背景：半透明深森绿遮罩（#2A5A14 @ 35%，比 v1 更浅，不压迫）。

**角色设计原则**：卡皮巴拉改为**线描简笔风格**——均匀 2px 轮廓线，暖棕色平涂填充，无彩铅纹理，无复杂阴影。配件极简化：每个场景最多 1 个配件（蒲公英 OR 小鸭子，二选一）。情绪表达保留：月牙眼/圆点眼/星星眼，嘴形简单变化。

**留白布局规范**（核心原则）：
- 屏幕任意区域的 UI 密度不超过 60%（至少 40% 空白）
- 所有相邻 UI 元素间距 ≥ 24px 设计坐标
- 卡片/弹窗内 padding ≥ 20px（四边均匀）
- 文字行高 ≥ 1.6 倍字号
- 按钮高度 ≥ 60px（设计坐标），左右 padding ≥ 24px

**屏幕适配与安全区规范**：与 v1 完全一致（顶部安全区 ≥ 160px，胶囊按钮右侧留空 x > 562px，底部安全区 ≥ 148px 设计坐标）。
