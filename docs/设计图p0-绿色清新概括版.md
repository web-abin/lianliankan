# 🎨 游戏视觉全案：春日草地与卡皮巴拉（全场景版）

本中心文档定义了《卡皮巴拉对对消》微信小游戏的视觉规范、设计语言及所有核心场景的 AI 生图 Prompt 模板。

## 一、 全局设计语言 (Global Design System)

### 1. 核心视觉风格
* **风格定义**：春日草地彩铅水彩儿童绘本风。
* **画面特征**：可见斜向彩铅笔触纹理、不规则手绘轮廓线、水彩边缘晕染。
* **构图规范**：竖屏 9:16 中心对称构图，整体传递轻松治愈的生命力感。

### 2. 标准色盘 (Color Palette)
| 用途 | 颜色名称 | 十六进制色码 | 质感描述 |
| :--- | :--- | :--- | :--- |
| **主背景** | 嫩草绿 | #AADC3C | 明亮清新，叠加 30° 斜向彩铅笔触 |
| **背景次色** | 浅草黄 | #C8E850 | 模拟受光面，增加画面层次 |
| **天空/水体** | 天空蓝 | #5BC8E8 | 清澈水彩感，通透呼吸感 |
| **主按钮/强调** | 暖珊瑚橙 | #FF7A45 | 核心交互，厚色彩铅块面 |
| **次按钮/辅助** | 蒲公英黄 | #F5D840 | 辅助功能，活泼明亮 |
| **描边/文字** | 深森林绿 | #2A5A14 | 3-5px 手绘不规则抖动线条 |
| **危险/血量** | 暖红 | #E8553A | 如草莓色，手绘感圆润 |
| **UI 底板** | 云朵白 | #FFFEF2 | 纸张原色，带微弱纤维纹理 |
| **角色主色** | 辅助棕 | #8B6040 | 卡皮巴拉暖棕色，显毛绒感 |

### 3. 关键组件规范
* **描边 (Outlines)**：严禁机械直线。所有 UI 元素使用深森林绿 (#2A5A14) 手绘感勾线，线条有粗细变化，末端圆帽，无锐角。
* **按钮 (Buttons)**：彩铅厚块 3D 效果。左上角白色彩铅高光，右下角 4px 深色阴影偏移，点击带 scale(0.95) 回弹。
* **弹窗 (Popups)**：云朵白圆角矩形 (Radius 28px)。顶部必须有卡皮巴拉 IP 形象探出或坐立，配合 Bounce 入场动画。
* **角色 (Character)**：卡皮巴拉为大圆头、小豆眼（月牙/星星眼），身体可见彩铅质感。常伴有小鸭子、蒲公英、蝴蝶。

## 四、 技术实现建议
* **动画**：弹窗采用 300ms 的 `Bounce` 弹性放大入场。
* **交互反馈**：按钮按下时执行 `scale(0.95)` 并播放轻微的水彩渲染反馈。
* **特效系统**：消除时产生的草叶与星粒应符合彩铅颗粒感，不宜过分平滑。
