# 🎨 游戏视觉全案：春日草地与卡皮巴拉（彩铅绘本风）

本中心文档定义了游戏所有界面的视觉规范、设计语言及 AI 生图 Prompt 模板。

---

## 一、 全局设计规范 (Global Design System)

### 1. 核心视觉风格
* **风格定义**：春日草地彩铅水彩儿童绘本风。
* **画面特征**：可见斜向彩铅笔触、不规则手绘轮廓线、水彩边缘晕染。
* **构图规范**：竖屏 9:16 中心对称构图。

### 2. 标准色盘 (Color Palette)
| 用途 | 颜色名称 | 十六进制色码 | 质感描述 |
| :--- | :--- | :--- | :--- |
| **主背景** | 嫩草绿 | #AADC3C | 明亮清新，叠加 30° 斜向笔触 |
| **高光背景** | 浅草黄 | #C8E850 | 模拟受光面，增加画面层次 |
| **天空/水体** | 天空蓝 | #5BC8E8 | 清澈水彩感，通透呼吸感 |
| **主按钮/强调** | 暖珊瑚橙 | #FF7A45 | 核心交互，厚色彩铅块面 |
| **次按钮/辅助** | 蒲公英黄 | #F5D840 | 辅助功能，活泼明亮 |
| **危险/血量** | 暖红 | #E8553A | 如草莓色，手绘感圆润 |
| **UI 底板** | 云朵白 | #FFFEF2 | 纸张原色，带微弱纤维纹理 |
| **描边/文字** | 深森林绿 | #2A5A14 | 3-5px 手绘不规则抖动线条 |

### 3. 组件规范
* **描边 (Outlines)**：严禁机械直线。所有元素使用深森林绿 (#2A5A14) 手绘感勾线，线条有粗细变化，末端圆帽。
* **按钮 (Buttons)**：厚块 3D 效果。左上角白色彩铅高光，右下角 4px 阴影偏移。
* **弹窗 (Popups)**：云朵白圆角矩形 (Radius 28px)。顶部必须有卡皮巴拉 IP 形象探出或坐立。
* **角色 (Character)**：卡皮巴拉为暖棕色 (#8B6040)，大圆头、小豆眼。

## 四、 技术实现建议
* **动效**：入场采用 Bounce 弹性动画。
* **点击反馈**：按钮按下时 scale(0.95) 并有回弹效果。
* **消除特效**：消除时迸发绿色草叶与暖橙星粒，伴随“棒！”等手绘飘字。
