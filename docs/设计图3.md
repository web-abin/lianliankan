这是一款治愈系手绘水彩风格的游戏界面设计，整体以「秋日森林」为主题：薄荷青天空、金黄落叶、暖琥珀大地三层叠构，辅以奶白圆润角色。风格参考绘本水彩插画（类似 ne.dear mind 系列），彻底告别沉重棕系，转向清新、轻盈、治愈的自然色调，更贴近休闲小游戏的松弛气质。

---

## 全局设计语言

**风格定义**：秋日绘本水彩萌 Q 风，类 Sanrio / Suatelier 插画感。柔和渐变色块、低饱和自然色系、强调空气感与留白，传递清晨秋日散步的温柔松弛感。

**色彩体系**（低饱和水彩处理，整体加白提亮）：

| 用途 | 色值 | 说明 |
|---|---|---|
| 天空背景 | #B8DDD0 | 薄荷青，清晨雾气感 |
| 浅天/渐变上层 | #D4EDE6 | 极淡薄荷白 |
| 地面/大地 | #C8973A | 秋日暖琥珀 |
| 地面浅色 | #E0B86A | 落叶金黄地 |
| 主按钮色 | #F09830 | 暖橙金，果实感 |
| 次按钮/强调 | #F5C040 | 金黄叶色 |
| 描边色 | #3A4E30 | 深森林绿（代替深棕，更清新） |
| 辅助薄荷绿 | #7CC5A8 | 嫩草/叶脉绿 |
| 辅助天蓝 | #89C4D8 | 秋日晴空蓝 |
| 秋叶金 | #E8B820 | 银杏/枫叶金黄 |
| 暗色文字 | #3A4E30 | 深森林绿，替代深棕 |
| 白色/浅底文字 | #FFFFFF | 按钮文字 |
| 危险/减血红 | #D95B4A | 柔化暖红 |
| 金币色 | #F0C030 | 秋金 |
| 弹窗底色 | #F5FBF8 | 极淡薄荷奶白 |

**描边规范**：统一使用 3px 深森林绿（#3A4E30）圆润描边，末端圆帽，转角圆弧过渡，无锐角。描边较原棕色方案更细、更轻，整体感觉更透气。

**按钮规范**：主按钮「秋果 3D」效果——顶部弧形淡金高光（白色半透明渐变条）、底部投影（#2D3E20 偏移 3px）、整体圆角矩形（radius 24-32px）；点击 scale(0.95) + 投影收缩反馈。

**弹窗规范**：弹窗主体为薄荷奶白圆角矩形（#F5FBF8，radius 32px，深森林绿描边 3px），顶部必须有 IP 角色（白熊/卡皮巴拉秋日版）探出顶边。入场动画：scale 从 0.6 弹性放大至 1.05 再回落 1.0，约 300ms。背景：半透明深森林绿遮罩（#3A4E30 @ 50%）。

**角色设计原则**：
- **主角「秋熊」**（参考图同款白熊）：极简几何圆体，奶白色身体（#F5F0E8），小豆眼，双颊淡粉晕染，憨态可掬；手持扫帚/持落叶/打伞等秋日动作。
- **配角**：小灰刺猬（抱着橡果）、两只金色小鸭（排队）、远景小羊驼。
- 情绪表达：月牙眼（享受）/ 圆点眼（机警）/ 星星眼（兴奋）/ 泪眼（委屈）。

**背景构图**：三层叠构——上层薄荷青天空（渐变 #B8DDD0 → #D4EDE6）+ 中层远山（#8CC5A8 柔绿）+ 下层琥珀大地（#C8973A）；天地交界处可有散落的秋叶粒子（金黄色，半透明，轻飘动画）。
