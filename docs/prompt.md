我想做一个连连看的微信小游戏，功能玩法和一般的连连看小游戏完全一致，我这款小游戏主打以美术和音乐取胜，以下是我想的大体的功能清单，没想到的地方你可以帮我补充一下：

整体风格：治愈卡通
1. 首页： 圈子好礼、喊人、添加桌面、每日奖励、开始游戏、每日挑战、商店、主题、排行榜
<!-- 以 OpenSpec 为准：llk-wechat-shell spec -->
- 圈子好礼：弹窗提示，加入游戏圈有奖励，送金币
- 喊人： 弹窗提示，邀请好友得血量
- 添加桌面：弹窗提示，每天从桌面进入可以领金币
- 每日奖励，弹窗提示，每日签到领金币，连续签到天数越多，领取的金币越多，上限是7天的签到奖励，再多也和7天的奖励一样了
- 开始游戏：逐渐闯关，显示目前的关卡，例如第4关，点击进入第4关，不能重玩前面关卡
- 每日挑战，直接进入一个比较难的关卡
- 商店：进入页面，可以购买音效、购买首页卡皮巴拉的形象、购买道具等，如果金币不够时弹窗提示：在游戏中通过关卡时，可以获得金币，买血量
- 主题：可以切换游戏的连连看图集，例如水果系列、厨具系列、动物系列，通过闯关获得，或者每日挑战获得
  <!-- 以 OpenSpec 为准：主题仅通过成就解锁，见 llk-achievements-themes spec -->
- 排行榜：查看好友排行榜和世界排行榜

- 设置： 音效、震动、音乐、问题反馈

2. 游戏界面：
<!-- 以 OpenSpec 为准：llk-gameplay-core spec -->
显示当前关卡数、当前关卡进度
游戏主区域（连连看区域），要有点击选中的特效、连错特效、连接消除特效
既有特效又有音效
道具：提示（获得3个提示）、刷新（打乱所有砖块）、消除（消除一组砖块）
这些道具暂时都先通过分享获得，等我开通流量主之后全部改成看激励广告获得
<!-- 以 OpenSpec 为准：分享+3，刷新+1，消除+1；激励视频预留配置入口，见 llk-economy-store spec -->

有关卡效果时需要显示一个蒙层的新手引导，例如消除之后，砖块靠左贴

闯关不限时，点击设置里的放弃挑战时需要有一个挽回弹窗： 内容提示
 当前已完成 进度
 再努力一下就要过关了
 再试试
 放弃挑战（回到首页）

 没有闯关成功提示，闯关成功后直接进入下一关，重新铺砖块，每次铺设砖块都要有特效和音效（包括第一关）

- 设置： 音效、震动、音乐、重玩本关、放弃挑战
